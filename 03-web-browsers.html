Le site internet caniuse.com permet de vérifier rapidement la compatibilité d'une technologie.

Moteurs de rendu HTML

Les navigateurs web peuvent comprendre 3 langages :

  1. Le HTML : Permet de structurer les données (carrosserie, essieu...)
  2. Le CSS : Permet de mettre en forme les données (peinture, enjoliveurs...)
  3. Le JS : Permet d'ajouter du dynamisme (phares, essuie-glaces, radio...)

Le moteur pourrait être comparé au langage côté serveur (PHP, Java, Python...) permettant ainsi à la voiture d'avancer.
