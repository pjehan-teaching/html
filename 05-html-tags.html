Les balises HTML

Les balises

  • Une balise est un élément de texte (un nom) encadrée par le caractère inférieur (<) et le caractère supérieur (>).
  • Les balises HTML ne sont pas sensibles à la casse.
  • Les balises HTML fonctionnent par paire afin d'agir sur les éléments qu'elles encadrent (balise ouvrante et balise fermante précédée par le caractère « / »)
  • Les balises HTML peuvent parfois être uniques et ne pas contenir d'information : la balise <br/> ou <img src="./logo.png"/>
  • Les balises HTML peuvent contenir des attributs (par exemple l'attribut src de la balise img).

Il existe plus de 100 balises HTML. Nous ne verrons donc que les plus utilisées.

Le code minimal

<!DOCTYPE html>
<html lang="fr-FR">
    <head>
        <title></title>
        <meta charset="utf-8">
    </head>
    <body>
    </body>
</html>
  • <!DOCTYPE html> pour indiquer la version HTML
  • <html></html> pour englober la page
  • <head></head> pour définir les informations de la page
  • <title></title> pour définir le titre de la page
  • <body></body> pour englober le corps de la page

La balise head

<head>
    <title>Titre de ma page</title>
    <meta charset="utf-8">
    <meta name="description" content="Apprendre le HTML et le CSS" />
    <meta name="keywords" content="HTML,CSS" />
    <meta name="author" content="Tim Berners-Lee" />
    <meta name="language" content="fr-FR" />
    <meta name="robots" content="noindex" />
</head>

La balise head

<script type="text/javascript" src="app.js" />

Il est préférable de mettre les appels JavaScript avant la balise <body/> Si l'on utilise l'attribut defer, le code JavaScript sera exécuté après le chargement de la page

Inline VS Block

Certaines balises sont de type inline et d'autres de type block.

Inline

<span><span/> <i><i/> <strong><strong/>

Block

<div><div/> <h1><h1/> <p><p/>

Commentaires

Il est possible (et recommandé) de documenter son code HTML en utilisant des commentaires :

<!-- Titre de niveau 1 -->
<h1>Mon titre</h1>
Ne pas mettre d'informations sensibles dans les commentaires HTML !

La sémantique

Certaines balises ont un impact sur le référencement (SEO) et d'autres n'en ont pas.

Par exemple, les balises <div> ou <span> ne sont pas sémantiques tandis que les balises <h1> ou <strong> ont une signification (respectivement titre de niveau 1 et texte important).

Pour afficher un texte important en gras, on pourrait être tenté d'utiliser la balise HTML <b>. Cependant, cette balise n'a pas de valeur sémantique pour les moteurs de recherche et il sera donc préférable d'utiliser la balise <strong>. De manière générale, la mise en forme sera gérée par le code CSS le HTML devra se focaliser sur la sémantique.

Titres

Il est possible de créer des titres de plusieurs niveaux en utilisant les balises suivantes :

  • <h1></h1>
  • <h2></h2>
  • <h3></h3>
  • ...

En fonction du navigateur, le titre sera affiché plus ou moins grand.

Il est possible de définir plusieurs h1 sur une même page

Titres (exemples)

<h1>Niveau 1</h1>
<h2>Niveau 2</h2>
<h3>Niveau 3</h3>

Conteneurs

Il est préférable d'utiliser des balises sémantiques comme conteneurs :

  • <header></header>
  • <footer></footer>
  • <main></main>
  • <section></section>
  • <article></article>
  • <nav></nav>
  • <aside></aside>

Ces balises sont toutes de type block et n'ont pas d'impact visuel sur la page.

Ces balises sont apparues avec la version 5 du langage HTML

Paragraphes

La balise <p></p> permet de contenir un paragraphe.

La balise <br> permet d'effectuer un retour à la ligne (très rarement utilisée).

Un paragraphe ne peux contenir que des éléments de type inline

Mise en avant du texte

La balise <strong> permet de marquer un texte comme important.

Il existe également les balises :

  • <em> : emphasized
  • <mark> : marked / highlighted
  • <b> : bold
La balise <b> ne doit pas être utilisée pour la mise en forme (CSS)

Mise en avant du texte

<strong>Strong</strong>
<em>Emphasized</em>
<mark>Marked</mark>
<b>Bold</b>
Strong Emphasized Marked Bold

Listes

La balise <ul> (unordered list) permet de créer une liste à puces.

La balise <ol> (ordered list) permet de créer une liste à puces ordonnées.

La balise <li> (list item) permet de créer un élément d'une liste et doit impérativement se trouver dans un balise <ul> ou <ol>.

Les balises <dl>, <dt> et <dd> peuvent être utilisées pour une liste de mots avec leur définition.

Il est également possible de définir la forme de la puce (disc, circle...) en CSS

Listes (exemple)

<ul>
    <li>un</li>
    <li>deux</li>
    <li>trois</li>
</ul>
  • un
  • deux
  • trois

Tableaux

La balise <table> permet de créer un tableau.

Les balises <thead>, <tbody> et <tfoot> permettent d'identifier l'en-tête, le corps et le pied du tableau.

La balise <tr> permet de créer une ligne.

Les balises <th> et <td> permettent de créer respectivement une cellule d'en-tête et une cellule du corps d'un tableau.

Il ne faut pas utiliser les tableaux pour positionner des éléménts

Tableaux (exemple)

<table>
    <thead>
    <tr>
        <th>Nom</th>
        <th>Prénom</th>
        <th>Note</th>
    </tr>
    </thead>
    <tbody>
    <tr>
        <td>John</td>
        <td>Doe</td>
        <td>14</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>Doe</td>
        <td>15</td>
    </tr>
    </tbody>
    <tfoot>
    <tr>
        <td>-</td>
        <td>-</td>
        <td>14.5</td>
    </tr>
    </tfoot>
</table>

John Doe 14
Jane Doe 15
- - 14.5
Les attributs colspan et rowspan permettent de regrouper des cellules

Images

La balise <img> permet de créer une image.

L'attribut src permet de définir la source (fichier à afficher).

L'attribut alt permet de définir le texte à afficher si l'image ne se charge pas correctement.

Toutes les images doivent avoir un attribut alt

Liens

La balise <a> permet de créer un lien.

L'attribut href permet de définir la destination.

L'attribut target permet de définir la manière donc la redirection sera effectuée (la valeur _blank permet par exemple d'ouvrir dans un nouvel onglet).

Les serveurs Apache affichent le fichier index.html par défaut

Liens

Il est possible de créer des liens vers une section d'une page (ancre) en ajoutant un "#" au début de l'attribut href et en ayant un élément avec l'id correspondant dans la page.

<nav>
    <ul>
        <li><a href="#presentation">Présentation</a></li>
        <li><a href="#nos-services">Nos services</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>

<section id="#presentation">
    ...
</section>

Liens

Il est également possible d'ouvrir automatiquement le client email ou téléphone lorsque l'on clique sur un lien :

<a href="mailto:contact@monsite.com">M'écrire un mail</a>
<a href="tel:0123456789">0123456789</a>

Liens (exemple)

<nav>

        <li><a href="http://www.google.com" target="_blank">Google</a></li>
