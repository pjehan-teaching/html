Les outils de développement

Editeur de code

Les éditeurs de code sont des logiciels permettant d'éditer du code plus facilement.

  • Coloration syntaxique
  • Indentation
  • Auto-complétion
  • Refactoring
  • ...

Outils de développement des navigateurs

La touche F12 permet d'ouvrir une fenêtre de développement et d'avoir accès à un grand nombre d'outils permettant d'inspecter une page web.

  • Voir et modifier le code HTML, CSS ou JS
  • Débugger le code JS
  • Afficher les messages de la console JS
  • Exécuter du code JS
  • Tester un site responsive
  • Inspecter les échanges avec le serveur
  • Optimiser les performances
  • Afficher les cookies
  • ...
