SGML (Standard Generalized Markup Language)

  • Langage de description à balises
  • Composé en trois parties :
    1. DTD (structure du document)
    2. Feuilles de style (mise en forme du document)
    3. Données
  • XML est une version simplifiée de SGML

Doctype (ou DTD)

  • Permet de spécifié le type d’un document SGML ou XML
  • N’est utilisé par les navigateur pour reconnaitre la version HTML utilisé mais PAS pour la validation du code (il pourra être utilisé à l’aide par exemple des outils en ligne du W3C)

HTML (HyperText Markup Language)

  • Langage basé sur le SGML créé en 1991 par Tim Berners-Lee
  • Pas un langage de programmation
  • Langage de balisage d’hypertexte

XHTML (EXtensible HyperText Markup Language)

  • Langage respectant les normes du XML
  • Plus strict que le langage HTML
  • La déclaration DOCTYPE est requise
  • L'attribut xmlns de la balise <html> est requis
  • Les balises doivent être correctement imbriquées, toujours fermées et être en lowercase.
  • Les attributs doivent être en lowercase, leur valeur entre quote et doivent obligatoirement avoir une valeur.
<!DOCTYPE HTML PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <title></title>
    </head>
    <body>
    </body>
</html>
