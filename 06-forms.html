Les formulaires

Créer un formulaire

Un formulaire est délimité par une balise <form></form>.

L'attribut method permet de définir la méthode HTTP (GET, POST, PUT...) utilisée pour l'envoi des données du formulaire.

L'attribut action permet de définir la page qui sera chargée de traiter l'envoi des données du formulaire.

Vous pouvez également utiliser une balise <input> ou <button></button> de type submit pour envoyer les données.

Zone de texte

Vous pouvez utiliser la balise auto-fermante <input> pour définir un champ de saisie.

L'attribut type permet de définir le type de données à saisir (text, email, date, number...)

L'attribut placeholder permet d'indiquer la saisie attendue dans le champ.

L'attribut required permet d'indiquer que le champ est requis.

Liste des types d'input L'attribut name est indispensable pour gérer l'envoi des données

Zone de texte

Libellé

Les libellés permettent d'indiquer à l'internaute ce qu'il doit saisir dans un champ de formulaire.

La balise HTML <label></label> permet de créer un libellé.

L'attribut for permet d'associer un libellé avec l'id dans champ de formulaire.

Lorsque l'internaute clique sur un label, le focus se fait sur le champ de formulaire correspondant

Zone de texte multiligne

La balise <textarea></textarea> permet de créer une zone de texte multiligne.

Les attributs rows et cols permettent de modifier la taille de la zone de texte.

Contrairement aux inputs, les textarea n'ont pas d'attribut value

Choix multiples

La balise <input> de type checkbox permet de créer une case à cocher.

La balise <input> de type radio permet de créer un bouton radio.

Ces éléments peuvent être regroupés en leur attribuant un attribut name identique.

L'attribut checked permet de sélectionner une option par défaut.

L'attribut value permet de définir la donnée qui sera envoyée.

Listes déroulantes

La balise <select></select> permet de créer une liste déroulante.

La balise <option></option> permet de créer un choix de la liste déroulante.

L'attribut value des balises <option></option> permet de définir la donnée qui sera envoyée.

La balise <select></select> peut également être une balise à choix multiples en ajoutant l'attribut multiple.

Regroupement de champs

Vous pouvez utiliser les balises <fieldset></fieldset> et <legend></legend> pour regrouper des champs de formulaire.
